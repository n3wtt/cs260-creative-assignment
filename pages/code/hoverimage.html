Here's some css that will create a grid, and than change the images to be the same size and grow on hover!

.grid-container { position: relative; margin-left: 120px; margin-top: 30px; display: grid; grid-row-gap: 10px; grid-template-columns: auto auto auto; padding: 10px; } img { max-width: 100%; max-height: 100%; } .grid-item { border: 1px solid rgba(0, 0, 0, 0.8); padding: 5px; transition-duration: 2s; height: 200px; width: 200px; } .grid-item:hover { transform: scale(2, 2); }
